# Postman Collections

Use this directory for Postman collections that exercise API workflows.

## Standard

- Prefer Postman Collection Format `v2.1.0`.
- Keep environments and secrets out of committed collections.
- Collections should map to OpenAPI operations when possible.
- Use examples for onboarding, smoke tests and manual verification.

## Files

- `repository.template.postman_collection.json`: minimal collection template.
- `repository.forge.postman_collection.json`: future Repository Forge workflow collection.
