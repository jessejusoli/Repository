# RAG Contracts

Use this directory for retrieval-augmented generation system profiles.

## Standard

- Define corpus owner, data class and retention.
- Document ingestion, chunking, embedding, retrieval and evaluation.
- Include privacy, prompt-injection and hallucination controls.
- Keep source data and indexes out of Git unless explicitly approved.

## Files

- `rag-profile.template.yml`: profile template for a RAG system.
- `repository.rag-profile.yml`: RAG profile for this repository documentation,
  including pillars, standards, catalogs and contracts.
