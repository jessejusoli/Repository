# Dados

## Objetivo

Gerencia analytics, big data, BI e possivelmente ML/IA. Ferramentas: BigQuery, Snowflake para storage; Tableau/Power BI para dashboards; TensorFlow ou Scikit-learn para machine learning.

## Quando Aplicar

Aplique este pilar quando um projeto declarar responsabilidade relacionada a Dados.
Se o pilar nao se aplicar, registre a justificativa no README do projeto ou no checklist universal.

## Evidencias Esperadas

- README do projeto com pilares aplicaveis.
- Registro em `catalog/projects.yml` quando houver projeto concreto.
- Decisoes, contratos, testes, automacoes ou runbooks conforme o item exigir.
- Revisao durante pull request usando `docs/checklists/universal-project-readiness.md`.
