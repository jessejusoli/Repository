# Branch Protection And Rulesets

## Objetivo

Orientar quando e como aplicar `branch-protection-rulesets` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: github-hosted-projects.

## Criterios De Adocao

- Main branch requires pull requests and checks.
- Force push and deletion are blocked for protected branches.
