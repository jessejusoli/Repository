# Changelog Standard

## Objetivo

Orientar quando e como aplicar `changelog-standard` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: released-projects.

## Criterios De Adocao

- User-facing changes are recorded.
- Release notes can be produced from the changelog.

## Artefatos Esperados
