# CODEOWNERS

## Objetivo

Orientar quando e como aplicar `codeowners-standard` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: all-projects.

## Criterios De Adocao

- Sensitive paths require owner review.
- Owners point to real users or teams before enforcement.
