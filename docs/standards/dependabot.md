# Dependabot

## Objetivo

Orientar quando e como aplicar `dependabot` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: github-hosted-projects.

## Criterios De Adocao

- Dependency update automation is enabled for supported ecosystems.
- Pull request volume is grouped or limited.

## Artefatos Esperados
