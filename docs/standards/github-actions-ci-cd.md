# GitHub Actions CI/CD

## Objetivo

Orientar quando e como aplicar `github-actions-ci-cd` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: github-hosted-projects.

## Criterios De Adocao

- CI validates repository health and project-specific checks.
- Workflow permissions are least privilege.

## Artefatos Esperados

- `.github/workflows/ci.yaml`
- `.github/workflows/cd.yaml`
