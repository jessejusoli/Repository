# License And SPDX

## Objetivo

Orientar quando e como aplicar `license-spdx` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: all-projects.

## Criterios De Adocao

- Owner chooses a real license before public distribution.
- Use SPDX identifiers where applicable.

## Artefatos Esperados

- `LICENSE or LICENSE.template.md`
- `catalog/license-options.yml`

## Links Oficiais

- [Documentacao oficial](https://spdx.dev/learn/handling-license-info/)

## Como Usar No Repository

- Consulte este pocket guide para decidir aplicabilidade.
- Registre evidencia no README do projeto, catalogo, contrato ou checklist correspondente.
- Use o link oficial para detalhes completos, exemplos extensos e mudancas de versao.
