# OpenAPI

## Objetivo

Orientar quando e como aplicar `openapi` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: http-apis.

## Criterios De Adocao

- HTTP APIs publish an OpenAPI contract before consumers depend on them.
- Contract includes paths, schemas, errors and auth expectations.

## Artefatos Esperados

- `contracts/openapi/*.yaml`
