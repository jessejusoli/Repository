# README Standard

## Objetivo

Orientar quando e como aplicar `readme-standard` sem copiar a documentacao oficial para dentro do Repository.

## Quando Aplicar

Aplica-se a: all-projects.

## Criterios De Adocao

- README gives orientation, commands, ownership, status and links.
- README avoids duplicating long external docs.
